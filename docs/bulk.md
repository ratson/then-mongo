# Bulk

## Methods

### bulk.find(query)

Returns `{upsert(), remove(), removeOne(), update(updateObj), updateOne(updateObj), replaceOne(updObj)}`

### bulk.insert(doc)

Returns `void`

### bulk.tojson()

Returns `Object`

### bulk.toString()

Returns `Object`

### bulk.execute()

Returns [`Promise`](https://www.promisejs.org/api/)
